## 7) Security + performance notes for Netlify static deployment

- Add strict security headers in `_headers`
- Use `defer` for JS and `loading="lazy"` for non-critical images
- Prefer compressed WebP images and explicit width/height
- Keep third-party scripts minimal and audited

## 🔍 TROUBLESHOOTING

### Logo Not Showing?

**Check:**
1. Is `images` folder uploaded?
2. Is `logo.png` inside `images` folder?
3. Check file path in HTML: `<img src="images/logo.png">`

**Quick Fix:**
- Make sure folder structure is: `yoursite.com/images/logo.png`

### Colors Look Wrong?

**Check:**
1. Browser cache — Clear it and refresh (Ctrl+Shift+R)
2. CSS file loaded — View page source, check `:root` variables
3. File uploaded correctly — Re-upload HTML files

### Header Looks Broken on Mobile?

**Check:**
1. Viewport meta tag present in `<head>`:
   ```html
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   ```
2. Mobile browser — Try different mobile browser
3. Cache — Clear mobile browser cache

### WhatsApp Links Not Working?

**Update this in all HTML files:**
```html
Old: https://wa.me/919591555095
New: https://wa.me/91YOURNUMBER
```
